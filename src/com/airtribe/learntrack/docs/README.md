1. Project description:
    LearnTrack is a console-based Student and Course Management System built using Core Java.
    The application allows administrators to manage Students, Courses and Enrollments through a menu-driven 
    command-line interface.
   
    The program using OOPs principles like Encapsulations, Inheritance, and basic Polymorphism.
    It follows layered approach with clear separation between entities, repositories, service 
    and user interface.

    All data is stored in memory using collections, and the system supports operations such as
    adding, viewing, deactivating entities and manage enrollments.
